# login Service
## Spring Framework를 이용한 Web Application 공부

<br>

login Service 구성 

* HashMap을 이용하여 메모리 저장 Repository
* validation 적용
* 접근 제한 필터링
  - filter를 통한 제한 필터링
  - interceptor를 통한 접근 제한 필터링 ( 현재 사용 )

* session을 통한 로그인 여부 확인
* view template
  - thymeleaf 사용
  - bootstrap 사용
